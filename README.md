# Predicting-Insurance-Claim-Amount
Medical insurance companies estimate insurance charges based on personal and medical information such as age, BMI, smoking habits, and number of children. Predicting insurance claim amounts helps companies make better pricing decisions and risk assessments.
In this project, a Linear Regression model was developed to predict medical insurance charges using the Medical Cost Personal Dataset. The project also analyzes how factors like BMI, age, and smoking status affect insurance costs.

This project successfully implemented a Linear Regression model to predict medical insurance charges using personal and medical data.
Key Insights
•	Smoking has the greatest impact on insurance charges. 
•	Age and BMI are also important factors influencing medical costs. 
•	Linear Regression can effectively estimate insurance claim amounts. 
•	Data visualization helped identify important relationships between variables. 
Overall, the project demonstrated how machine learning can be used in the healthcare and insurance industry for cost prediction and decision-making.
